Formulários

1. Gerar ID Externo

Fechaste um negócio em partilha com um colega? Vais precisar de um ID externo, para poder pedir a validação da minuta do contrato ou agendar a escritura. Gera o ID aqui!

2. Validação de angariação

Usa este link para pedires a validação de uma nova angariação, após teres colocado no SouKW. Certifica-te antecipadamente que toda a informação inserida, descrição, fotos e vídeos estão corretamente colocados antes de pedires a validação, e que tens a documentação devidamente identidicada para anexar.

3. Apoio processual

Preenche este formulário caso precises de esclarecer alguma questão, se pretendes fazer uma alteração numa angariação ou no caso de precisares da emissão de um documento.

4. Pedido de Contrato
ou Agendamento de Escritura

Fechaste negócio? Parabéns! Este é o formulário que deves utilizar. Seja para pedir um novo CPCV para uma angariação tua, para validar um CPCV de uma partilha com um colega (no caso de teres apenas o comprador), para um contrato de arrendamento ou no caso de agendamento de escritura (de um CPCV já celebrado, ou no caso de um novo negócio que vá direto para escritura). Não precisas de carregar novamente a documentação do imóvel caso seja uma angariação da Coastline! Se se tratar de um imóvel externo, gera primeiro o ID no formulário 1.

5. Envio Documentação de Negócio

Se os teus clientes assinaram um contrato ou realizaram uma escritura, é aqui que tens de carregar cópia destes documentos. Lembra-te de anexar todos os comprovativos de pagamento no caso do sinal entregue num CPCV.

6. Pedido Fatura

Concluíste um negócio e queres pedir a emissão de fatura? Usa este formulário quer se destine a emitir fatura para o teu cliente proprietário ou para uma agência com quem partilhaste negócio! Se possível, pede que te enviem o comprovativo de pagamento quando o realizarem, para mais facilmente identificarmos.

7. Pagamento Conta Corrente

Tens valores a receber na tua conta corrente? Verifica o valor que tens a receber, emite a fatura ou recibo correspondente e preenche este formulário. Vais precisar também de indicar o teu IBAN.

8. Esclarecimento Administrativo

Precisas de ajuda? Se a tua questão for relacionada com as nossas plataformas informáticas, portais imobiliários, minutas ou documentos, ou se tens alguma dúvida relativa à tua conta corrente, preenche este formulário!

9. Esclarecimento Comercial

Se procuras orientação, este é o formulário certo: preenche-o caso tenhas uma questão relativa a formação, jurídica, se queres falar sobre crescimento ou se precisas ajuda para uma negociação.



Todos os teus pedidos terão resposta até 48h - 2 dias úteis. Qualquer situação urgente contacta um elemento da Equipa de Liderança, estamos cá para ajudar a fechar negócios!
